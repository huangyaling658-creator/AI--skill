# daily-plan Skill

**Invoked by:** User or Claude

## Description

为特定用户生成符合其个人风格的今日计划，并推送到 Apple 备忘录。

当用户说"帮我生成今天的计划"、"我今天要做什么"、"今天计划"、"帮我安排今天"、"我说一下今天的想法"、"今天打算干"等类似表达时，立即触发此 skill。

用户会用语音或文字告诉你今天的大方向，你需要结合用户的固定风格画像生成完整计划，并通过脚本写入 Apple 备忘录。每次只要涉及"今日计划"或"帮我安排今天"就必须用这个 skill。

---

## 核心原则

这个 skill 的目标是生成一份看起来像用户自己写的计划，不是模板化的 AI 输出。

> 计划 ＝ 日记碎碎念 ＋ 时间块任务表。缺少任何一部分都不算完整。
